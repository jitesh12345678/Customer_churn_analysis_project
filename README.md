# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior in a telecom company using the Telco Customer Churn dataset. The goal is to identify key factors influencing customer attrition and provide actionable insights to improve customer retention.

## Tools Used
- Microsoft Excel (Data Cleaning)
- MySQL (Data Analysis)
- Power BI (Dashboard Development)

## Dataset
- Dataset: Telco Customer Churn
- Records: 7,043 Customers
- Features: 33 Columns

## Project Workflow
1. Data Cleaning in Excel
   - Checked duplicates
   - Handled missing values
   - Validated data types
   - Standardized categorical fields

2. SQL Analysis in MySQL
   - Total Customers
   - Churned Customers
   - Churn Rate
   - Churn by Gender
   - Churn by Contract Type
   - Churn by Internet Service
   - Top Churn Reasons
   - Average Monthly Charges Analysis

3. Power BI Dashboard
   - Executive Overview
   - Customer Insights
   - Churn Analysis
   - Customer Segmentation
   - CLTV Analysis

## Key Insights
- Month-to-month contract customers show the highest churn.
- Customers with higher monthly charges are more likely to churn.
- Internet service type significantly impacts churn behavior.
- Contract duration strongly influences customer retention.
- Pricing and service-related issues are major churn drivers.
